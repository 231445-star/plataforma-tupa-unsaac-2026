## Descripción del proyecto
El proyecto consiste en desarrollar una plataforma web para optimizar la consulta, registro y seguimiento de los procedimientos administrativos incluidos en el TUPA de la UNSAAC.

La plataforma permitirá consultar trámites, requisitos, costos y plazos de atención. Además, facilitará el registro de solicitudes, el seguimiento del estado de los trámites y la gestión administrativa mediante reportes y notificaciones.

## Funcionalidades principales
- Consulta de procedimientos administrativos.
- Visualización de requisitos, costos y plazos.
- Registro de solicitudes de trámites.
- Seguimiento del estado de solicitudes.
- Gestión de documentos digitales.
- Notificaciones al usuario.
- Reportes administrativos.

## Estructura del repositorio
- `database/`: script SQL de la base de datos del sistema.
- `docs/informe/`: documentación del proyecto.
- `docs/capturas_git/`: evidencias del uso de Git y GitHub.
- `interfaces/`: pantallas y recursos visuales del sistema.
- `prototipo/`: prototipos dinámicos de la plataforma.
